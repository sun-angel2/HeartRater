# ❤️ iQOO Watch GT 心率助手 (HeartRate Monitor)

一个轻量级的开源工具，用于在 Windows 电脑、手机和网页上实时显示 iQOO Watch GT (或其他标准蓝牙心率设备) 的心率数据。

![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Android%20%7C%20Web-blue)
![Python](https://img.shields.io/badge/Python-3.9%2B-green)
![Status](https://img.shields.io/badge/Status-Stable%20v4.1-success)

## ✨ 主要功能

*   **🖥️ 桌面 HUD 悬浮窗**：在 Windows 桌面以半透明 HUD 风格显示实时心率，支持拖动。
*   **🎨 动态颜色**：心率数值颜色随强度自动变化（绿 -> 黄 -> 橙 -> 红）。
*   **🌐 远程 Web 共享**：
    *   **IPv6 直连**：生成 IPv6 链接，手机/平板可通过浏览器远程查看（支持 4G/5G 直连）。
    *   **MQTT 云同步**：通过 GitHub Pages 实现跨网络、跨地域的心率直播。
*   **📱 手机端支持**：无需安装 APP，通过手机浏览器即可连接手表并上传数据。
*   **🛡️ 稳定连接**：内置蓝牙断连自动重试机制（不死鸟模式）。

## 🚀 使用方法 1：Windows 电脑版

适用于：你在电脑前，想把心率显示在屏幕上（如打游戏、直播 OBS）。

### 1. 下载与安装
如果你不想配置 Python 环境，请直接在 [Releases](../../releases) 页面下载最新的 `.exe` 文件。

如果你是开发者：
```bash
git clone https://github.com/你的用户名/你的仓库名.git
pip install customtkinter bleak flask
python hr.py
```

### 2. 操作步骤
1.  **打开蓝牙**：确保电脑蓝牙开启，手表点亮屏幕并处于心率界面。
2.  **扫描连接**：点击软件界面的 **“① 扫描设备”** -> 选择手表 -> **“② 连接”**。
3.  **开启远程**：点击 **“开启 Web 服务”**，复制链接发给手机即可远程查看。
4.  **自定义**：可以拖动悬浮窗位置，或在软件面板调整字体大小。

> **注意**：首次运行如果 Windows 防火墙弹窗，请务必勾选 ✅专用网络 和 ✅公用网络。

## 🌐 使用方法 2：全平台网页版 (推荐)

适用于：不想开电脑，只想用手机连接手表，分享给朋友看。

**无需安装任何软件**，直接访问以下链接：

| 角色         | 操作链接                                                              | 说明                                                    |

3.  保持手机屏幕常亮。

## 🛠️ 常见问题 (Q&A)

**Q: 电脑版开启 Web 服务后，手机打不开链接 (502 Bad Gateway)？**
*   **原因**：通常是手机或电脑开了代理软件，或者是防火墙拦截。
*   **解决**：
    1.  手机关闭 WiFi，使用 **4G/5G 流量** 访问。
    2.  电脑关闭 Clash/v2ray 等代理软件，或者配置分流规则。
    3.  检查 Windows 防火墙入站规则是否放行了 `8088` 端口。

**Q: 网页版点“连接”没反应？**
*   请确保使用的是 Chrome 浏览器，且网站是 HTTPS 协议（GitHub Pages 默认支持）。iOS 设备目前对此功能支持较差，建议使用安卓。

**Q: 心率颜色代表什么？**
*   **< 100**: 🟢 绿色 (热身/轻松)
*   **100 - 120**: 🟡 黄色 (燃脂)
*   **120 - 140**: 🟠 橙色 (有氧)
*   **> 140**: 🔴 红色 (无氧/极限)
